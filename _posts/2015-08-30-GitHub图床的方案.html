博客图片存储的几种可选方案。

图片放在哪里

GitHub：随仓库托管，与 Jekyll 站点一起管理，适合博客内图片。

Dropbox + Pancake：图片可外链，Pancake 地址固定；后缀需小写 .png。缺点：Pancake 访问受限，刷新过频会暂时不可用；Dropbox 免费空间仅 2G。

七牛云：试用后手工操作尚可；命令行工具和批量 API 有局限，月度请求数容易超限。

百度相册、Skydrive：类似 Windows Live Writer 的联机相册，自定义不便。

相册需求

希望有一个文件夹存放几十张图片，配置一个网页：显示缩略图，点击可放大或新选项卡查看原图。
