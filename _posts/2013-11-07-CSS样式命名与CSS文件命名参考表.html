DIV+CSS 常用命名与文件命名的参考表（转自 divcss5.com）。

网页公共命名

布局：#wrapper、#container、#content、#layout

结构：#head / #header、#foot / #footer、#nav、#subnav、#menu、#submenu

侧栏：#sideBar、#sidebar_a、#sidebar_b

主内容：#main

功能：#tag、#msg、#searchInput、#searchBar、#copyright、#logo 等

通用：.left、.right、.center、.news、.download、.banner、.tab

电子贸易相关

.products、.products_prices、.products_description、.publisher、.screenshot、.faqs、.keyword、.blog、.forum

CSS 文件命名

master.css / style.css（主样式）、module.css（模块）、base.css（基础）、layout.css（布局）、themes.css（主题）、font.css（字体）、forms.css（表单）、mend.css（补丁）、print.css（打印）
