GitHub Pages 可绑定自定义域名，但同一仓库一般只能绑定一个域名。

绑定步骤

较易成功的是 blog.hypergroups.cn 这类子域名。若要绑定 www.hypergroups.cn 或 hypergroups.cn，在仓库根目录的 CNAME 文件中填写 www.hypergroups.cn。在域名商处添加 A 记录，指向 GitHub 的 IP。网上常见有 192、207、113 等，可能随地区和时期不同，可先 ping hypergroups.github.io 获取当前 IP 作为参考。

常见问题

绑好后有时本地仍打不开，可能因 DNS 生效较慢。可咨询他人能否访问；若能访问则多为本地缓存问题。Windows 下执行：

ipconfig /flushdns

清理 DNS 缓存后再试。
