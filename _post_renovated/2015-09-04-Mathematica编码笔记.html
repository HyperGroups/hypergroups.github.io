CP936、Unicode、UTF-8、WindowsANSI 是 Mathematica 中常见的文本编码。

常见问题

ExportString 编码设置无效：某些导出格式下 CharacterEncoding 可能不起作用，需用其他方式预处理。

FileTemplateApply、FilePrint：文件编码需显式指定。

C 编译器 Debug 信息、JLink Java Console：对中文支持有限。

Read/ReadList：导入时需指定 CharacterEncoding。
