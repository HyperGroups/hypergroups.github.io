本文记录了在 Mathematica 中多种方式实现 HelloWorld 的尝试：Mathematica 内置、C 编译器（CCompiler）、Java、JLink、以及 Scala 等。

概要

Mathematica：使用 HelloWorld 或 Print@HelloWorld 输出；也可用 GUIRunModal 创建简单 GUI 弹窗。

C：通过 CCompiler 相关功能调用 C 代码。

Java / JLink：Needs["JLink`"] 与 InstallJava[] 加载 JVM，用 LoadJavaClass、JavaNew 加载并实例化 Java 类，通过 objHelloWorld@main[{}] 调用 main 方法。JLink 的 Java Console 对中文支持有限。

Scala 等：对于没有直接 Link 的语言，可用 RunProcess 或 ! 等外部进程方式调用。
